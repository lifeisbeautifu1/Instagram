# Instagram

Instagram Clone build with Next.js

## Stack

Next.js Next-Auth React.js Typescript TailwindCSS Firebase Recoil

## Features

1. Google Auth.
2. Create posts.
3. Comments + likes on posts.
4. Image uploading with firebase storage.
